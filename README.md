## 📊 Sobre o projeto

O **SIGECON** é um sistema web completo desenvolvido para simular um ambiente corporativo real de gestão de contratos, fornecedores e indicadores estratégicos.

O sistema foi criado com foco em **eficiência operacional, organização de dados e análise de desempenho**, aplicando conceitos de sistemas empresariais utilizados no dia a dia corporativo.

Este projeto faz parte do meu portfólio como desenvolvedora em transição de carreira para a área de tecnologia, unindo experiência administrativa com desenvolvimento de software.

---

## ⚙️ Funcionalidades

- 🔐 Sistema de autenticação (login com rotas protegidas)
- 📄 Gestão completa de contratos
  - Cadastro
  - Visualização
  - Controle de status
- 🏢 Gestão de fornecedores
- 📊 Dashboard com indicadores estratégicos
- 📈 Visualização de dados com gráficos dinâmicos
- ⏰ Alertas de vencimento de contratos
- 📤 Exportação de relatórios em PDF
- 🔄 Navegação entre módulos via React Router
